**hr**

A command line utility to print a horizontal rule.

# Usage

```sh
$ hr
------------------------------------------------------------------------------

$ hr -r '*' -w 20 
********************

$ hr -o '/** ' -c ' */' -r '*' -w 20
/** ************* */
```
